# HiraganaLearner

> An electron-vue project made for learning hiragana

#### Build Setup

``` bash
# use NPM if you don't have Yarn installed

# install dependencies
npm install
yarn

# run developer at localhost:9080
npm run dev
yarn run dev

# build electron application for production
npm run build
yarn run build


```
